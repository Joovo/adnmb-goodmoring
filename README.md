# adnmb-goodmorning
每日抓取A岛"咕猫宁"微博，并自动评论

每五分钟查询“A岛匿名版”微博最新动态，在每日"咕猫宁"微博下实现自动评论：

“📢 在最美的年华，做最喜欢的事情的说，别辜负了美好时光。借时光之手，暖一处花开，借一方晴空，拥抱梦想的说。早安的说！”

# 实现思路

- 抓取微博接口，持续查询最新的微博，获取"咕猫宁"的微博地址
- 利用pyppeteer直接登陆
- 调整schedule定时任务，合理分配网络资源
